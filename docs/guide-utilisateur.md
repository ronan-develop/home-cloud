# Guide d'utilisation — HomeCloud

Bienvenue sur votre espace HomeCloud, un cloud personnel et privé conçu pour stocker, organiser et partager vos fichiers et photos en toute simplicité.

Ce guide vous explique pas à pas comment utiliser toutes les fonctionnalités disponibles.

## 1. Première connexion — par ici pour commencer

### Se connecter

Ouvrez votre navigateur (Chrome, Firefox, Safari…) et tapez l'adresse de votre espace personnel dans la barre en haut :

```
https://prenom.lenouvel.me
```

*(remplacez `prenom` par votre prénom)*

Vous arrivez sur la page de connexion. Saisissez :
- votre **adresse e-mail** (celle communiquée à Ronan)
- votre **mot de passe provisoire** : `mot_de_passe`

Puis cliquez sur **Se connecter**.

---

### Changer votre mot de passe — à faire dès la première connexion

Le mot de passe fourni au départ est provisoire. **Il est important de le changer** pour sécuriser votre espace.

Voici comment faire :

1. Une fois connecté, cliquez sur votre **prénom** (ou l'icône de profil) en haut à droite de l'écran.
2. Cliquez sur **Paramètres**.
3. Faites défiler jusqu'à la section **Changer le mot de passe**.
4. Remplissez les trois champs :
   - *Mot de passe actuel* : `mot_de_passe`
   - *Nouveau mot de passe* : choisissez-en un personnel (au moins 8 caractères)
   - *Confirmer le nouveau mot de passe* : retapez-le
5. Cliquez sur **Enregistrer**.

Votre nouveau mot de passe est actif immédiatement. La prochaine fois que vous vous connecterez, utilisez ce nouveau mot de passe.

> **Conseil** : choisissez un mot de passe que vous n'utilisez nulle part ailleurs — une suite de 3 ou 4 mots dont vous vous souvenez fait très bien l'affaire (ex. `soleil-chat-montagne`).

> **Mot de passe oublié ?** Pas de panique — sur la page de connexion, cliquez sur *Mot de passe oublié*. Vous recevrez un e-mail avec un lien pour en choisir un nouveau.

## 2. Découvrir l'interface

Une fois connecté, vous arrivez sur le **tableau de bord**.

L'interface est divisée en deux parties :

- **Sur ordinateur** : un menu à gauche liste les sections (Mes fichiers, Galerie, Albums…). Le contenu s'affiche à droite.
- **Sur téléphone ou tablette** : une barre de navigation apparaît en bas de l'écran, avec les mêmes sections accessibles d'une pression.

En haut à droite, une icône vous permet de basculer entre le **mode clair** et le **mode sombre**, selon vos préférences.

## 3. Mes fichiers — stocker et organiser

C'est ici que vous retrouvez tous vos fichiers, rangés dans des dossiers comme sur votre ordinateur.

### Envoyer des fichiers pour la première fois

Voici comment envoyer votre première photo ou votre premier document :

1. Dans le menu, cliquez sur **Mes fichiers**.
2. Vous arrivez sur votre espace, vide pour l'instant.
3. Cliquez sur le bouton **Envoyer** (ou le **+**) en haut de la page.
4. Une fenêtre s'ouvre : sélectionnez un ou plusieurs fichiers sur votre ordinateur, puis cliquez sur **Ouvrir**.
5. Une barre de progression apparaît — attendez qu'elle atteigne 100 %.
6. Votre fichier est maintenant dans votre espace. Si c'est une photo ou une vidéo, elle apparaîtra aussi dans la **Galerie** dans quelques minutes.

**Astuce — glisser-déposer :** si vous préférez, vous pouvez directement faire glisser un fichier depuis votre bureau ou votre explorateur de fichiers vers la page. Lâchez-le n'importe où dans la zone centrale — il s'envoie automatiquement.

> Tous les types de fichiers sont acceptés (photos, vidéos, documents PDF, musique…), à l'exception des fichiers exécutables (.exe, .bat…).

---

### Créer un dossier pour s'organiser

Ranger vos fichiers dans des dossiers vous permettra de les retrouver facilement. Par exemple : un dossier *Vacances 2024*, un dossier *Documents importants*, etc.

1. Cliquez sur le bouton **Nouveau dossier**.
2. Tapez le nom du dossier.
3. Appuyez sur **Entrée** ou cliquez sur **Créer**.

Vous pouvez créer des dossiers dans des dossiers, autant que vous le souhaitez.

---

### Renommer un fichier ou un dossier

1. Repérez le fichier ou le dossier à renommer.
2. Cliquez sur les **trois points** (⋯) à droite de son nom.
3. Choisissez **Renommer**.
4. Effacez l'ancien nom, tapez le nouveau, puis validez.

---

### Déplacer un fichier dans un dossier

**Par glisser-déposer :** cliquez sur le fichier, maintenez le bouton de la souris enfoncé, faites-le glisser vers le dossier de destination, et relâchez.

**Par le menu :** cliquez sur les **trois points** → **Déplacer**, puis choisissez le dossier de destination dans la liste.

---

### Télécharger un fichier sur votre appareil

1. Cliquez sur les **trois points** à côté du fichier.
2. Cliquez sur **Télécharger**.
3. Le fichier se sauvegarde dans le dossier *Téléchargements* de votre ordinateur ou téléphone.

---

### Supprimer un fichier ou un dossier

1. Cliquez sur les **trois points** à côté du fichier ou dossier.
2. Cliquez sur **Supprimer**.
3. Confirmez la suppression lorsque la question vous est posée.

> ⚠️ La suppression est définitive — il n'y a pas de corbeille.

## 4. La Galerie — vos photos et vidéos

La Galerie regroupe automatiquement **toutes les photos et vidéos** que vous avez envoyées, quelle que soit leur localisation dans vos dossiers. Vous n'avez rien à faire — elles apparaissent d'elles-mêmes.

> **Bon à savoir** : après l'envoi d'une photo, il peut s'écouler quelques minutes avant qu'elle apparaisse dans la Galerie avec sa vignette. C'est normal — elle est traitée en arrière-plan.

---

### Voir une photo en grand — première fois

1. Dans le menu, cliquez sur **Galerie**.
2. Vous voyez toutes vos photos sous forme de petites vignettes.
3. Cliquez sur l'une d'elles pour l'ouvrir en plein écran.
4. Pour passer à la photo suivante : cliquez sur la **flèche droite** à l'écran, ou appuyez sur la touche **→** de votre clavier.
5. Pour revenir en arrière : **flèche gauche** ou touche **←**.
6. Pour fermer : cliquez sur la **croix** en haut à droite, ou appuyez sur la touche **Échap**.

---

### Lancer un diaporama

Une fois une photo ouverte en plein écran, cliquez sur le bouton **Diaporama** (icône de lecture ▶). Les photos défilent automatiquement. Cliquez à nouveau pour arrêter.

---

### Filtrer et trier

En haut de la Galerie :
- Bouton **Photos** : affiche uniquement les photos.
- Bouton **Vidéos** : affiche uniquement les vidéos.
- Menu **Trier** : classez par date, par nom ou par taille selon vos préférences.

---

### Photos RAW

Si vous prenez vos photos avec un appareil photo reflex ou hybride, vous utilisez peut-être le format RAW (CR2, CR3, NEF, ARW, DNG…). HomeCloud les reconnaît et en affiche automatiquement un aperçu — vous n'avez pas besoin de les convertir. Le fichier RAW d'origine est conservé intact pour que vous puissiez le télécharger à tout moment.

## 5. Les Albums — créer des sélections

Les albums vous permettent de regrouper des photos selon vos envies — par événement, par personne, par lieu… — sans les déplacer de leur dossier d'origine. Une même photo peut apparaître dans plusieurs albums.

### Créer votre premier album

1. Dans le menu, cliquez sur **Albums**.
2. Cliquez sur **Nouvel album**.
3. Donnez un nom à votre album (ex. : *Noël 2024*) et validez.
4. L'album est créé — il est vide pour l'instant.

---

### Ajouter des photos à un album

1. Allez dans la **Galerie**.
2. Survolez une photo avec votre souris : une petite **case à cocher** apparaît en haut à gauche de la vignette.
3. Cochez toutes les photos que vous voulez ajouter.
4. En bas de l'écran, cliquez sur **Ajouter à un album**.
5. Choisissez l'album dans la liste, puis confirmez.

---

### Réorganiser les photos d'un album

Dans un album, cliquez sur une photo et faites-la **glisser** à l'endroit souhaité. L'ordre est sauvegardé automatiquement.

---

### Choisir la photo de couverture

La couverture est la grande photo qui représente l'album dans la liste.

1. Ouvrez l'album.
2. Survolez la photo que vous souhaitez mettre en couverture.
3. Cliquez sur l'icône **étoile** qui apparaît.

## 6. Partager avec quelqu'un

HomeCloud propose deux façons de partager un fichier, un dossier ou un album.

### Partager avec une personne par e-mail — première fois

Imaginons que vous voulez partager un dossier de photos de vacances avec un ami :

1. Dans **Mes fichiers**, repérez le dossier à partager.
2. Cliquez sur les **trois points** à côté de son nom.
3. Cliquez sur **Partager**.
4. Saisissez l'adresse e-mail de votre ami et confirmez.

Ce qui se passe ensuite :
- Si votre ami n'a pas de compte HomeCloud, il reçoit un **e-mail d'invitation** avec un lien pour accéder directement au dossier — sans inscription de sa part.
- Si votre ami a déjà un compte, il reçoit une **notification par e-mail** avec un lien direct.

Votre ami peut **consulter et télécharger** ce que vous avez partagé, mais il ne peut pas modifier ni supprimer vos fichiers.

---

### Partager par lien — pour envoyer par SMS ou WhatsApp

Vous voulez partager un album avec toute la famille sans leur demander de créer un compte ? Utilisez le partage par lien.

1. Ouvrez l'album (ou le fichier, ou le dossier).
2. Cliquez sur les **trois points** → **Partager par lien**.
3. Choisissez combien de temps le lien sera valable :

| Durée | Usage conseillé |
|-------|----------------|
| 1 jour | Partage express, pour aujourd'hui |
| 7 jours | Partage d'une semaine |
| 30 jours | Partage prolongé |
| Permanent | Sans limite de temps |

4. Copiez le lien qui apparaît et envoyez-le par SMS, WhatsApp, e-mail…

Toute personne qui clique sur ce lien pourra consulter la ressource, **sans avoir besoin de compte**.

Pour **supprimer le lien** et couper l'accès : retournez sur les **trois points** → **Partager par lien** → cliquez sur la corbeille à côté du lien existant.

> **Confidentialité** : les pages accessibles via un lien de partage ne sont pas indexées par Google ou les autres moteurs de recherche.

## 7. Gérer les invités

La section **Invités** liste toutes les personnes avec qui vous avez partagé du contenu.

Depuis cette page, vous pouvez :

- **Créer un compte invité** directement, en saisissant son e-mail et son prénom — utile pour préparer un partage à l'avance.
- **Renommer** un invité pour le retrouver plus facilement dans la liste.
- **Supprimer** un invité : son accès est coupé immédiatement à tout ce que vous aviez partagé avec lui.

## 8. Rechercher un fichier

Vous ne retrouvez plus un fichier ? Utilisez la barre de **recherche** en haut de l'interface.

1. Cliquez sur la barre de recherche (icône loupe 🔍).
2. Tapez quelques lettres du nom du fichier ou du dossier.
3. Les résultats apparaissent au fur et à mesure que vous tapez.
4. Cliquez sur le résultat pour y accéder directement.

---

## 9. Votre compte

### Modifier votre nom affiché

1. Cliquez sur votre **prénom** en haut à droite.
2. Cliquez sur **Paramètres**.
3. Dans le champ **Nom**, modifiez votre prénom ou votre nom complet.
4. Cliquez sur **Enregistrer**.

### Changer votre mot de passe

1. Dans **Paramètres**, faites défiler jusqu'à la section **Changer le mot de passe**.
2. Remplissez les trois champs : mot de passe actuel, nouveau mot de passe, confirmation.
3. Cliquez sur **Enregistrer**.

### Basculer en mode sombre

Cliquez sur l'icône ☀️ / 🌙 en haut à droite pour changer de thème. L'application suit par défaut la préférence de votre téléphone ou ordinateur.

## 10. En cas de problème

**Je ne reçois pas l'e-mail de réinitialisation de mot de passe**
→ Vérifiez votre dossier *Indésirables* ou *Spam*. Si l'e-mail n'arrive pas dans les 5 minutes, contactez Ronan.

**Mes photos n'apparaissent pas dans la Galerie après l'envoi**
→ C'est normal — elles sont traitées automatiquement en arrière-plan. Attendez 2 à 5 minutes puis rafraîchissez la page (touche **F5** sur ordinateur, ou tirez la page vers le bas sur mobile).

**Je ne retrouve pas un fichier**
→ Utilisez la barre de recherche en tapant quelques lettres de son nom.

**J'ai supprimé un fichier par erreur**
→ La suppression est définitive pour l'instant. Contactez Ronan dès que possible — il pourra peut-être le récupérer selon la situation.

**L'application semble lente ou ne répond plus**
→ Rafraîchissez la page (touche **F5** ou **Ctrl+R** sur ordinateur). Si le problème persiste, contactez Ronan.

**Un lien de partage ne fonctionne plus**
→ Le lien a peut-être expiré. Demandez à la personne qui vous l'a envoyé d'en générer un nouveau.

## 11. Naviguer dans les dossiers imbriqués — le fil d'Ariane

Lorsque vous entrez dans un dossier, puis un sous-dossier, un **fil d'Ariane** apparaît en haut de la page. Il ressemble à ceci :

```
🏠 > Vacances > Été 2024 > Photos
```

Chaque étape est cliquable. Cliquez sur un nom pour revenir directement à ce niveau, sans avoir à appuyer plusieurs fois sur le bouton retour.

- Cliquez sur l'icône **maison** 🏠 pour revenir à la racine de vos fichiers (le niveau le plus haut).
- Cliquez sur n'importe quel dossier intermédiaire pour y retourner directement.

---

## 12. Sélectionner plusieurs fichiers à la fois

Vous pouvez agir sur plusieurs fichiers en même temps — par exemple pour tous les supprimer ou les déplacer en une seule opération.

1. Dans **Mes fichiers**, survolez un fichier avec la souris : une **case à cocher** apparaît.
2. Cochez tous les fichiers que vous souhaitez sélectionner.
3. Une barre d'actions apparaît en bas de l'écran — choisissez **Supprimer** ou **Déplacer**.
4. Confirmez l'action.

> Sur mobile, appuyez longuement sur un fichier pour entrer en mode sélection multiple.

---

## 13. Déposer un fichier directement dans un sous-dossier

Vous n'avez pas besoin d'ouvrir un dossier avant d'y envoyer des fichiers. Depuis n'importe quelle vue de **Mes fichiers** :

1. Faites glisser votre fichier depuis votre ordinateur.
2. Survolez le dossier de destination — il se met en surbrillance pour indiquer qu'il est prêt à recevoir le fichier.
3. Relâchez — le fichier est directement envoyé dans ce dossier.

## 14. Consulter les informations d'une photo (métadonnées EXIF)

Les appareils photo enregistrent automatiquement des informations dans chaque photo : la date et l'heure de prise de vue, le modèle d'appareil, et parfois les coordonnées GPS du lieu où la photo a été prise.

HomeCloud lit ces informations et les affiche pour vous.

1. Ouvrez une photo en plein écran depuis la **Galerie**.
2. Un panneau d'informations est visible à côté ou en dessous de la photo.
3. Vous y trouverez (si disponibles) :
   - 📅 La **date et l'heure** de la photo
   - 📷 Le **modèle d'appareil** (ex. : *iPhone 15*, *Canon EOS R6*…)
   - 📍 Les **coordonnées GPS** du lieu de prise de vue

> Ces informations ne sont pas disponibles pour les photos RAW pour l'instant — c'est une amélioration prévue.

## 15. Accéder à un contenu partagé avec vous (côté invité)

Si quelqu'un vous a partagé un fichier, un dossier ou un album, vous recevez un **e-mail de notification**.

**Si vous n'avez pas de compte :**

1. Ouvrez l'e-mail reçu.
2. Cliquez sur le lien dans l'e-mail — votre compte invité est créé automatiquement.
3. Vous accédez directement au contenu partagé.
4. Pour vous connecter la prochaine fois, rendez-vous sur votre adresse HomeCloud personnelle et utilisez votre e-mail. Si vous n'avez pas encore de mot de passe, utilisez le lien *Mot de passe oublié* pour en créer un.

**Si vous avez déjà un compte :**

1. Connectez-vous normalement.
2. Le contenu partagé avec vous apparaît dans votre tableau de bord, dans une section dédiée.

En tant qu'invité, vous pouvez **consulter et télécharger** les fichiers partagés. Vous ne pouvez pas modifier, déplacer ni supprimer le contenu d'autrui.

---

## 16. Révoquer un partage — retirer l'accès à quelqu'un

Vous avez partagé un dossier avec quelqu'un et souhaitez lui retirer l'accès ?

### Retirer l'accès à une personne spécifique

1. Dans **Mes fichiers**, repérez le fichier ou dossier concerné.
2. Cliquez sur les **trois points** → **Partager**.
3. Dans la liste des personnes ayant accès, cliquez sur la **corbeille** à côté du nom de la personne.
4. Son accès est coupé immédiatement.

### Supprimer un lien de partage public

1. Repérez le fichier, dossier ou album concerné.
2. Cliquez sur les **trois points** → **Partager par lien**.
3. Cliquez sur la **corbeille** à côté du lien existant.
4. Le lien est invalidé — toute personne qui tente de l'utiliser verra une erreur.
